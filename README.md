## Identity & Cloud Security Engineering Program

## Overview

This portfolio documents hands-on projects focused on Identity and Access Management, Cloud IAM, cloud security analysis, DevSecOps, infrastructure as code, container security, and security documentation.

The goal of this program is to build realistic, job-ready projects that demonstrate how identity and access controls are designed, tested, secured, monitored, and documented in modern enterprise and cloud environments.

## Background Foundation

Before starting this cloud security program, I built an Active Directory lab covering users, groups, organizational units, domain-joined workstations, file-share permissions, and Group Policy.

That project helped establish my foundation in enterprise identity and access management before moving into AWS IAM and cloud security.

## Current Project

### 01 - AWS IAM Least Privilege Lab

This project focuses on designing and testing least-privilege access in AWS IAM. It includes IAM users, groups, policies, S3 access testing, CloudTrail evidence, access review documentation, and remediation of excessive permissions.
